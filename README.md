# 🎬 Video/Audio Transcriber

        A powerful desktop application that transcribes video and audio files using local Whisper models, with real-time display, SRT export, and continuous learning.

        ## ✨ Features

        - **Play any media** - Video (MP4, AVI, MKV) and Audio (MP3, WAV, FLAC)
        - **Real-time transcription** - Text appears every 2-3 seconds
        - **Local Whisper models** - 100% offline, complete privacy
        - **SRT export** - Save professional subtitles
        - **Continuous learning** - Gets smarter with every edit
        - **99+ languages** - Full language support

        ## 🚀 Quick Start

        ```bash
        # Install dependencies
        pip install -r requirements.txt

        # Run the application (Windows)
        run.bat

        # Alternative methods:
        run.ps1                    # PowerShell script
        ./.venv/Scripts/python.exe main.py  # Manual activation
        ```
