Video/Audio Transcriber

Local Whisper transcription for video and audio, with subtitle export and custom model support.

This guide covers installation, first use, custom model setup, transcription workflow, and common troubleshooting steps.

What This App Does

Play Media

Open local video or audio files and review them inside the app while transcribing.

Transcribe Offline

Use local Whisper models so audio stays on your machine.

Edit And Export

Review text, make corrections, and export subtitles as .srt or plain text.

Quick Start

  1. Install dependencies.
  2. Start the application.
  3. Load a media file.
  4. Choose a model.
  5. Start playback and transcription.
  6. Export the result when finished.
pip install -r requirements.txt
python main.py

Loading Media

Use File -> Open File... to load one item, or File -> Open Folder... to add multiple media files to the playlist.

The player tab contains playback controls, a seek bar, waveform display, volume control, and playback speed.

Models

The application supports two model styles:

Type Description Typical Source
Standard Whisper Preset model sizes such as tiny, base, small, medium, and large. OpenAI Whisper model downloads
Custom Hugging Face Whisper A local folder containing model and tokenizer files. Downloaded or fine-tuned Hugging Face Whisper model

Using A Custom Whisper Model

Open Models -> Model Manager..., then register or select your custom model folder.

A typical custom Whisper folder should include:

  • config.json
  • model.safetensors or pytorch_model.bin
  • tokenizer.json
  • tokenizer_config.json
  • processor_config.json
Important: if your folder has tokenize.json instead of tokenizer.json, rename or replace it. The processor typically expects tokenizer.json.

Transcription Workflow

  1. Load a media file.
  2. Choose the language from the Transcription -> Language menu, or leave it on auto.
  3. Select a model from the Models menu or the Model Manager.
  4. Start playback.
  5. Use Transcription -> Start Transcription if needed.
  6. Review text in the transcription panel.
  7. Edit lines and export subtitles.
Tip: sentence-aware chunking can produce cleaner segments for subtitle editing. You can toggle it from the Transcription menu.

Editing Font And Font Size

The transcription panel toolbar includes a font selector and a font size selector.

  • Use the font combo box to change the editor font while you review or edit subtitles.
  • Use the size field to increase or decrease the editor text size immediately.
  • These changes are saved and restored the next time you open the app.

Customizing Shortcut Keys

Open Settings -> Preferences... and use the Shortcuts tab.

  • You can reassign shortcuts for open file, export SRT, edit current line, find, start/stop transcription, zoom, fullscreen, and preferences.
  • Click in a shortcut field and press the key combination you want.
  • Leave a field empty if you want to disable that shortcut.
  • If the same shortcut is assigned to more than one action, the Preferences dialog will warn you and prevent saving until the conflict is fixed.
  • Click Apply or OK to save the updated shortcuts.

Exporting Results

Use these menu items after transcription:

  • File -> Export SRT... to create subtitle files.
  • File -> Export as Text... to save plain text.

Troubleshooting

Custom model does not load

  • Make sure the folder exists and contains the required files.
  • Check that transformers is installed for Hugging Face-style models.
  • Verify the tokenizer file is named tokenizer.json.

Playback is unavailable

  • Make sure VLC and the Python VLC bindings are installed.
  • Check the project’s vlc-help.txt if playback setup is incomplete.

Transcription is slow

  • Try a smaller Whisper model.
  • Use GPU/CUDA if available.
  • Lower playback speed while reviewing difficult audio.

Project Files Worth Knowing

  • README.md: quick overview and startup commands
  • vlc-help.txt: playback setup notes
  • models/: downloaded Hugging Face Whisper models
  • .cache/whisper or user cache: standard model storage
