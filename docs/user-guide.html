Video/Audio Transcriber Usage Guide

This application plays local media files, transcribes them with local Whisper models, lets you review and correct the text, and exports subtitles or plain text without sending audio to a cloud service.

This guide is written to match the current desktop application: menus, settings, playback controls, model management, correction-based learning, and common troubleshooting paths.

What The App Does

Local Playback

Open video or audio files, seek through them, control volume and playback speed, and review the waveform while you work.

Local Whisper Transcription

Transcribe with standard Whisper model sizes or a registered local Hugging Face Whisper model folder.

Subtitle Editing

Review live transcription or an existing SRT, edit lines, adjust sync, search text, and export as .srt or .txt.

The app is designed for offline work. Media stays on your machine, and model files are loaded from local storage.

Requirements

Required

  • Python environment with project dependencies from requirements.txt
  • Readable local media files such as .mp4, .avi, .mkv, .mov, .mp3, .wav, .flac, or .m4a
  • At least one Whisper model available locally or downloadable through the app

Recommended

  • VLC plus Python VLC bindings for the best playback support
  • A CUDA-capable GPU for faster transcription when available
  • Enough free disk space for models, waveform cache activity, and exported subtitles
If playback does not work, check vlc-help.txt. The transcription editor can still work with local files, but embedded playback depends on VLC support.

Install And Start

From the project folder, install dependencies and launch the application.

pip install -r requirements.txt
python main.py

You can also use the provided Windows launch scripts:

run.bat
run.ps1
On first use, start with the default small model unless you already know you want a larger or custom model.

Open Media And Build A Playlist

Open One File

Use File -> Open File... or the main toolbar open button. The app loads the file into the player and prepares the transcription panel for that media.

Open A Folder

Use File -> Open Folder... to add all supported media files from a folder into the playlist. This is useful when you need to work through a batch of recordings.

Playlist Features

  • Double-click a playlist item to load and play it.
  • Drag files or folders from Explorer into the app.
  • Use the playlist context menu to play, remove, or reveal a file in Explorer.
If a media file already has an .srt file with the same base name next to it, the app will load that subtitle file automatically when the media is opened.

Choose A Model

Standard Model Sizes

Use the Models menu to switch between standard Whisper sizes such as tiny, base, small, medium, and large. The app will use a local Hugging Face version when available and can download it if needed.

Custom Hugging Face Model

Open Models -> Model Manager... and register a local model folder. After it appears in the list, select it and click Use This Model.

Required Folder Contents Notes
config.json Model configuration
tokenizer_config.json Tokenizer setup is required
model.safetensors or pytorch_model.bin At least one model weights file must exist
tokenizer.json Strongly recommended for compatible tokenizer loading
processor_config.json Optional but commonly present
If a custom model folder is rejected, verify the filenames exactly. A common issue is a missing or incorrectly named tokenizer file.

Run A Transcription

  1. Open a media file.
  2. Choose the transcription language from Transcription -> Language, or leave it on auto.
  3. Select the model you want from the Models menu or Model Manager.
  4. Press play. In live mode, transcription begins while playback runs.
  5. Use Transcription -> Start Transcription if you need to trigger it manually.
  6. Use Transcription -> Stop Transcription or stop playback when you want to pause processing.

Sentence-Aware Chunking

The Sentence-Aware Chunking option groups output into more natural subtitle-sized sentences. This usually gives cleaner lines for editing and export.

  • Enable it when you care about subtitle readability.
  • Disable it when you prefer simpler chunk-by-chunk live updates.
  • If you change the setting while a file is already loaded, the app can prompt you to reload the file so the new behavior takes effect.

Playback Review

  • Use the seek bar or waveform to jump to a location.
  • Adjust playback speed for difficult speech.
  • Use volume control for quieter recordings.

Edit And Review Text

Editor Modes

  • Live mode: shows timestamped segments produced during transcription.
  • SRT mode: shows existing or exported subtitle entries in standard SRT structure.

Common Editing Actions

  • Use Edit -> Edit Current Line or the editor toolbar button to modify the active subtitle line.
  • Use Edit -> Find... to search the current transcript.
  • Use Sync Offset to shift all timestamps forward or backward in milliseconds.
  • Use Clear if you want to discard the current on-screen transcription content.

Font Controls

The transcription toolbar includes a font family selector and point-size control so you can make subtitle review more comfortable without affecting the media itself.

Every accepted line edit is also treated as a correction candidate for the learning system, which means your review work can later improve training data.

Export Results

Export As SRT

Use File -> Export SRT... or the editor toolbar export button. If you are in SRT mode, the existing subtitle entries are saved. If you are in live mode, the app converts the live segments into SRT entries first.

Export As Plain Text

Use File -> Export as Text... to save the current transcript as a .txt file.

Export happens from the transcription panel content that is currently loaded, so review the text and sync before saving final subtitles.

Preferences And Shortcuts

Open Settings -> Preferences... to access the full preferences dialog.

Transcription Tab

  • Default language
  • Chunk duration and overlap
  • Sentence-aware chunking
  • Beam size and temperature

Performance Tab

  • Compute device: auto, cpu, and cuda when available
  • Compute precision such as float32, float16, or int8 depending on hardware
  • CPU thread count
  • Model cache location and cache size display

Appearance Tab

  • Theme: dark or light

Correction Learning And Training

The app can record your subtitle corrections and use them as learning data for background training.

How Corrections Are Collected

  • Edit a subtitle line in live mode or SRT mode.
  • The app records original text, corrected text, timestamps, language, and the related file path when available.
  • Pending corrections appear in training statistics and status indicators.

Training Menu Actions

  • Train Now starts a manual training run if there are pending corrections.
  • View Training Stats shows correction totals, vocabulary growth, and recent training sessions.
  • Refresh Stats refreshes the correction and training indicators.
  • Clear All Corrections permanently removes saved correction data.
If Train Now says there is nothing to train, make a few subtitle edits first and then try again.

Troubleshooting

Documentation Does Not Open

  • Check that docs/user-guide.html exists in the project folder.
  • Make sure your desktop can open local HTML files with the default browser.

A Model Will Not Load

  • Confirm the selected model or custom folder exists locally.
  • For custom models, verify required filenames exactly.
  • If a standard model download fails, check dependency availability for Hugging Face downloads.

Playback Area Stays Blank

  • Install VLC and Python VLC support.
  • Review vlc-help.txt for setup notes.
  • Test with a simple local .mp4 or .mp3 file first.

Waveform Does Not Appear

  • Waveform generation depends on audio loading support.
  • Large files may take a little time to render.
  • Use View -> Show Waveform to confirm the waveform is enabled.

Transcription Is Slow

  • Switch to a smaller model size.
  • Use CUDA if available.
  • Choose a faster compute type such as int8 or float16 if your environment supports it.
  • Reduce review speed by lowering playback speed when accuracy matters more than throughput.

No Text Exports

  • Make sure the transcript panel actually contains segments or loaded SRT entries.
  • If live mode is empty, verify a model is loaded and playback was running while transcription was enabled.
